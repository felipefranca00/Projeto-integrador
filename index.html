Disciplinas




No desenvolvimento de um algoritmo, devemos definir com clareza e forma precisa o conjunto de regras
ou instruções que serão utilizadas para resolver um problema em específico. Portanto, antes de programar,
precisamos saber o que deve ser feito e planejar o passo a passo, ou seja, criar o algoritmo e avaliar se o
resultado obtido é a solução esperada. Entendido isso, então definimos uma linguagem de programação para
implementar nossos algoritmos. No primeiro semestre, você aprenderá a criar algoritmos utilizando o Python,
que é uma linguagem de programação de alto nível, simples e versátil, usada em diversas aplicações, que se
destaca pela sua simplicidade e legibilidade.


Nesta disciplina, você aprenderá a criar, construir e realizar a manutenção de websites e aplicações web.
Envolve trabalho tanto no front-end, a parte visual e interativa do site, quanto no back-end,
que é a infraestrutura interna. Serão utilizadas várias linguagens de programação, frameworks e tecnologias
para construir e melhorar experiências online. Durante o primeiro semestre, o aluno será capaz de realizar uma
One page como esta, utilizando as linguagens de marcação HTML, CSS em contribuição com Java Script. As
habilidades necessárias para se destacar nesta área abrangem uma vasta gama de conhecimentos técnicos,
desde linguagens de programação fundamentais como HTML, CSS e JavaScript, até o domínio de frameworks
complexos e sistemas de gerenciamento de banco de dados. A capacidade de adaptar-se às mudanças
tecnológicas e adotar novas metodologias é crucial para o sucesso e a inovação no campo do desenvolvimento
web.

Na disciplina de design digital, você aprenderá a criar elementos visuais de design orientados para os
meios digitais, e isso inclui vários formatos de trabalhos e atividades, como por exemplo, experiência do usuário
(UX Design), design de Interface do Usuário (UI Design), web design, design de Jogos, criação de anúncios digitais,
etc. O objetivo do design digital é criar soluções visuais, funcionais e interativas no meio digital.
A área tem como foco não só os aspectos gráficos e visuais, como tipografia, cores e outros elementos de design,
como também a experiência do usuário, interatividade, e no geral, a criação de interfaces digitais funcionais.

Esta disciplina consiste em técnicas que resultam em um projeto de desenvolvimento organizado, ou seja,
todos os aspectos da produção de um software, desde os estágios iniciais de especificação do sistema até sua
manutenção, depois que o sistema entrou em operação. Você aprenderá a descrever formalmente as atividades
a serem seguidas, fazer o levantamento de requisitos , que é uma das partes mais importantes do processo de
desenvolvimento, pois define quais são as necessidades e desejos do cliente para o projeto. Durante esse
processo, é fundamental que as etapas sejam seguidas, para garantir que o time esteja alinhado com as
expectativas do cliente e para o sucesso da implementação.

A modelagem de dados é uma etapa importante e essencial em qualquer projeto de desenvolvimento ou
manutenção de software. Atualmente, as organizações coletam grandes quantidades de dados de várias fontes
diferentes. No entanto, dados brutos não são suficientes. Nessa disciplina, você aprenderá a coletar, armazenar
e processar os dados para obter insights que possam orientar na tomada de decisões. Existem várias
tecnologias de banco de dados e ferramentas de processamento de dados que permitem uma análise eficiente.

Na disciplina de Sistemas Operacionais você vai descobrir como surgiram os primeiros computadores, o que
é um hardware, gerenciar recursos, tipos de estruturas além de entender como o sistema operacional do seu
computador faz esse diálogo entre homem e máquina, e o quanto ele é fundamental para garantir maior
eficiência e uma experiência facilitada para o usuário.











Profissões e Áreas de Atuação




Os profissionais de TI estão presentes em grandes empresas e em grande número.

Podem atuar em gestão de projetos, liderança e planejamento, desenvolvimento, consultoria e assistência

em indústrias, empresas de pequeno, médio e grande porte, ONG´s e setores públicos.

No Brasil, a média salarial de um desenvolvedor de software varia de acordo com a formação, região de atuação

e especializações.

É uma carreira promissora que garante remuneração acima da média.












 O gerente de TI está abaixo do diretor de tecnologia. Juntos, conduzem os projetos, organizando e revisando
processos junto às equipes de trabalho. É tarefa do gerente de TI identificar oportunidades de aplicação de novas
tecnologias no dia a dia da operação.

O desenvolvedor DevOps atua junto ao time de desenvolvimento de software. É o profissional que traz mecanismos
para dar mais confiabilidade e qualidade aos sistemas, desde a idealização do produto até a entrega ao cliente.

O cientista   tem conhecimentos sobre linguagens de programação, como SQL e Python, além de modelagem para
criar estruturas que dão coerências a um conjunto de dados.

O engenheiro desoftware é responsável pelo desenvolvimento e pela arquitetura de software,para web e para mobile.
Precisam desenvolver aplicações web (back-end ou front-end) e mobile (Android ou iOS), com diversos tipos de linguagem
de programação, frameworks e banco de dados.
